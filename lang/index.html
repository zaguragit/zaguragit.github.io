Constraints

  • The first syllable is always the stressed one (obviously, if ur singing, do whatever u want)
  • Syllables must follow the structure (C)V(n, m, l). The first consonant can only be omitted in first syllable
  • /t/, /k/, /s/ are used at the start of a word, unless followed by /a/, while /d/, /g/, /z/ are used elsewhere
  • The combinations ji/ij are not allowed

Numbers

Numbers (which are base 6), are written in arabic form. Do whatever you want, but "," won't be part of a number

Punctuation

⟨Untitled Conlang Experiment⟩ uses the same punctuation as most european languages

Syntax

⟨Untitled Conlang Experiment⟩ has an SVO (subject, verb, object) structure. Adjectives follow nouns, adverbs follow verbs. There's no distinction between adjectives, nouns & pronouns: the first one is treated as a nound, and the rest as modifiers

Numbers

The number system is seximal (base 6)

Single-digit numbers are represented with digit words:

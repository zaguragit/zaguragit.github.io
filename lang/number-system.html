Ilin Kil's Number System

main page: Ilin Kil

The number system is seximal (base 6)

Single-digit numbers & 10 are represented with these words:

Number Words
0 1 2 3 4 5 10 100
nil un ku mi pi ko zan zulo
Unexian Suffix
10000
-zol
Quick Syntax Table
Name Syntax Number
A nil|un|ku|mi|pi|ko [0, 5]
B A-zan A * 10
C B A B + A
D A|B|C
E D1 zulo D2
(D1 = 1 if missing)
(D2 = 0 if missing)
(zul å is optional if result is unambiguous) 		(D1 * 100) + D2
F A + -zol 10000 ^ A
G D|E
H G F H (G * F) + H

Examples

  • 43 = li-zan mi
  • 100 = zulo
  • 142 = un li-zan ku
  • 214 = ku un-zan li
  • 513 = ko zulo zan mi
  • 4251 = li-zan ku ko-zan un
  • 35 2042 = mi-zan ko unzol ku-zan li-zan ku
  • 300 0000 0000 = mi zulo kuzol

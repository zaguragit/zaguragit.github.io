Slab Launcher

Slab Launcher draws inspiration from YASAN & Ratio launcher, it has a simple tile-like grid of apps on the main page, with a dash, that shows you relevant information

Source Discord Telegram


General Features

  • Blur everywhere
  • Color palette generation from the wallpaper (different algorythms available)

Dash Features

  • Relevant notifications
  • Next alarm
  • Media player

Search Features

  • DuckDuckGo instant answers
  • App shortcuts
  • Contacts
